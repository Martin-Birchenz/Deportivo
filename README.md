# Sistema de gestión - Club Deportivo Nogoyá

Este proyecto es una solución y un paso hacia la transformación digital para el Club Deportivo Nogoyá.

# Gestión de usuarios y seguridad

- **Sistema de registro e inicio de sesión:** Sistema robusto con validación de credenciales.
- **Seguridad:** Encriptación de contraseñas y protección de rutas.
- **Persistencia:** Manejo de sesiones mediante Cookies.

# Módulo de socios

- **Asociación:** Formulario para que usuarios soliciten su carnet de socio.
- **Estado de Cuenta:** Visualización dinámica de cuotas sociales.
- **Gestión de Pagos:** Integración con datos bancarios y aviso de pago directo a WhatsApp.
- **Solicitud de Baja:** El usuario tiene la chance de dar de baja su asociación.

# Carrito de Compras

- **Tienda Online:** Catálogo de productos (indumentaria) del club.
- **Precios Dinámicos:** El sistema detecta el estado del socio y aplica un 20% de descuento en el carrito si está al día.
- **Gestión de Carrito:** Persistencia de productos seleccionados durante la navegación.

# Diseño e integración

- **Novedades:** Integración con la API de Instagram para mostrar las últimas publicaciones del club.
- **Diseño Profesional:** Interfaz responsive con Bootstrap.

# Herramientas y tecnologías

- **Frontend:** HTML, CSS, JavaScript, Bootstrap.
- **Backend:** Node.js, Express.
- **Base de Datos:** MySQL.
- **Herramientas de Desarrollo:** MySQL Workbench, Postman, Git y GitHub.
